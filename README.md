# Composants Vue.js — Props, Emits & Slots

Exercice de la séance 6 du cours C141 (Vue.js/Vuetify).

## Objectif

Compléter les `TODO` dans `App.vue` et `ChildComp.vue` pour pratiquer :

1. **Props** — passer des données du parent vers l'enfant (`defineProps`)
2. **Emits** — envoyer des événements de l'enfant vers le parent (`defineEmits`)
3. **Slots** — injecter du contenu HTML dans un composant enfant (`<slot>`)

## Utilisation

Ouvrir le [Playground Vue.js](https://play.vuejs.org/) et copier-coller le contenu des fichiers.

Ou importer directement ce dépôt dans le playground :
1. Ouvrir https://play.vuejs.org/
2. Cliquer sur l'icône GitHub en haut à droite
3. Entrer `fallinov/vue-composants-demo`
